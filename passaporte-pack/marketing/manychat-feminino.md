## 🛠️ Configuração no ManyChat

### Tags necessárias
- `arq_amante`, `arq_governante`, `arq_heroi` … (12 arquétipos)
- `amor_palavras`, `amor_tempo`, `amor_presente`, `amor_servico`, `amor_toque`
- `tag_comprador` (excludente)
- `box_relacionamento_oferecido`

### Custom Fields
- `result_url_quiz_arquetipo`, `result_url_quiz_amor`, `guide_url`, `coupon_url`, `external_id`

### Sub-flows reutilizáveis
- `[FLOW] Atendimento humano` → quando o usuário escolhe "tenho uma dúvida"
- `[FLOW] Lembrete 48h` → quando NÃO clicou no botão de checkout

### Filtros padrão (em todas as mensagens 2+)
```
SE has_tag = tag_comprador → não enviar
SE NOT clicked_button = checkout E hours_since > 48 → enviar lembrete leve
SE hours_since > 168 (7 dias) → encerrar sequência ou trocar para email
```

### Tom de voz padrão (não quebrar)
- Sempre que possível, escreva como uma amiga mais velha — não como vendedora.
- Use 1 emoji floral/sensorial por mensagem (🌹 🌅 ✨), nunca mais de 2.
- Frases curtas. Pausas com quebras de linha. Espaço para a leitora respirar.
- Nada de CAPS LOCK. Nada de "URGENTE!!!".
