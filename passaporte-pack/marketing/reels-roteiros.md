## 4. 📜 SÉRIE "CARTAS DE UMA MULHER A OUTRA"

> Sequência de 7 reels narrados (15-30s cada) com fundo simples, voz feminina cálida, texto na tela.

| Reel  | Tema                                                                                 |
| ----- | ------------------------------------------------------------------------------------ |
| #1    | "Carta para a mulher que diz sim quando quer dizer não" — perfil **Estável (S)**.     |
| #2    | "Carta para a mulher que termina tudo, menos consigo mesma" — perfil **Influente (I)** + **Amante**. |
| #3    | "Carta para a executiva que já não sabe descansar" — perfil **Dominante (D)** + **Heroína**. |
| #4    | "Carta para a perfeccionista que adia a própria vida" — perfil **Analista (C)** + **Criadora**. |
| #5    | "Carta para quem confunde amor com necessidade" — **Linguagem do Amor + Amante**.    |
| #6    | "Carta para quem é forte demais para chorar" — **Heroína / Eneagrama 8**.            |
| #7    | "Carta para a mulher que esqueceu de ser inteira" — **fechamento, CTA Passaporte**.   |

Cada carta termina em: _"Se essa carta tocou em algo seu, resgate seu Passaporte de Autoconhecimento no link da bio. Eu fiz para você."_

## 5. 🎯 NOTAS DE PERFORMANCE

- **Capa do post (feed):** sempre Azul Marinho `#0E1A33` ou Terracota `#A8522E` — alto contraste no feed feminino.
- **Hook nos primeiros 1.5s:** pergunta direta sobre dor relacional ou identidade (ex.: "Você se sente vista?").
- **Texto na tela:** máximo 10 palavras por frame. Use Cormorant Garamond Italic em destaque.
- **CTA fixo:** sempre "Comente [PALAVRA]" → automatize resposta no ManyChat com link + 10 créditos.
- **Frequência:** 4 reels/semana. 1 carrossel/semana. Stories diárias com bastidores.
